# 关于圣餐的几点澄清

一直以来不少兄弟姐妹咨询关于圣餐的问题，出于牧养的角度，终于有了下面这些文字。我们接下来就从圣经的角度、历史的角度、管理的角度来探讨这个问题。

一、从圣经的角度

1、圣餐是耶稣在和门徒享用最后的晚餐的时候设立的（太26:26-30；可14:22-26；路22:15-20；林前11:23-25），代表和他们立约，因着耶稣为他们流血牺牲，让他们罪得赦免。所以圣餐是为真正信的人或得救的人或基督徒设立的，是每一个基督徒必须遵守的圣礼。而主在颁布大使命的时候（太28:19-20），主吩咐门徒们要去使万民做主的门徒，奉父、子、圣灵的名给他们施洗。可16:16提到信而受洗的必然得救。神赐给教会确认一个人是否是基督徒的权柄（太18:17-18），教会有权柄和责任在确认一个人是基督徒之后才给他们施洗。所以一般来说，教会只给信而受洗的基督徒领圣餐。

2、圣餐的过程包括：1）拿起饼来祝谢。2）掰开。3）分给门徒并说：这是我的身体。4）拿起杯来祝谢。路22:20；林前11:25提到“饭后”，是指“吃完饼之后”的意思。5）递给门徒并说：这是我立约的血，为多人流出来的，使罪得赦。（可14:22-24；太26:26-28；路22:19-20；林前11：）。路22:19耶稣清晰地向门徒颁布守圣餐的命令，就是“你们应当如此行”。

3、圣餐的意义是为了纪念主耶稣（路22:19；路11:24-26），其核心是耶稣的死，等于是福音的核心的呈现。既然要守圣餐直到主的再来，所以圣餐的意义也应该包括主的复活和再来。

4、圣餐中的哪些要素体现了圣餐的意义呢？1）吃饼--象征耶稣的身体为我们破碎。2）喝杯。--象征主的宝血为我们流出。以及用血所立的新约。所以林前11:27提醒我们要省察自己是否按照圣餐的意义以正确的心态和态度来领受，否则就干犯了圣餐。因此有的人被神管教而患病，甚至有死的（林前11:30-32）。

5、什么是正确的心态或态度呢？1）承认自己的罪和不配，在主面前谦卑。耶稣是为了我们的罪上十字架的。如果说我们没有罪，就是自欺了（约一1:8）。2）承认耶稣为我们的罪舍了身体流出宝血，让我们的罪得赦免（约一1:9）。他就是我们的救主和生命（约6:51-58）。3）承认主是守约施慈爱的神，他一定会再来（林前11:25）。他用血和我们立的新约一定会实现（路22:20），他的应许永不落空，他给我们的盼望一定不会让我们羞愧（罗5:5）。他就是我们的主、我们的神和我们的王，他会永远和我们同在（来8:10-12；启21:3-7；彼前1:3-5；西1:21-23）。4）需要彼此等待，在敬畏和感恩中进行，个人省察自己的信仰，分辨是主的身体和宝血，而不是混乱无序或稀里糊涂，甚至酒醉（林前11:20-21；28-29；33-34）。

6、什么是错误的心态或态度呢？和上述的不一致的都是错误的，常见的错误还有：1）认清自己的罪才能领圣餐。其实我们从小到大的罪，甚至最近的罪，我未必完全记得，而且还有隐而未现的罪。2）悔改完最近所有的罪才能领圣餐。其实我们的罪永远是悔改不完的，所以才需要信耶稣。3）如果最近犯了一个严重的罪，就不能领圣餐。圣餐是主给每一个基督徒的命令。只要你还是教会认可的基督徒，还没有被开除（林前5:4,13），就可以领圣餐。保罗给哥林多教会写书信的时候，他们还有很多罪没有悔改（林前3:1-3）。

7、圣餐的环节是否必须要祝谢呢？不一定。因为这不是有象征意义的基本环节。我们需要告诉会众圣餐设立的过程，包括耶稣先祝谢了再掰饼等等，但不是必须走过耶稣当年走过的每一个环节。祝谢在当时有点相当于现在的谢饭，表达对神对天父赐给我们食物的感谢。耶稣以及他的门徒吃饭的时候通常都祝谢（太15:36；可8:6；路24:30；约6:11，23；徒27:35；林前10:30；提前4:4）。当时主餐和吃饭是在一起的，所以就一起谢饭了。现在我们的圣餐通常是单独拿出来纪念主的时间，所以未必需要走谢饭的程序。有人说必须要为圣餐祝福（太26:26；可14:22；可6:41），这样圣餐才能分别为圣，其实祝福就是祝谢的意思（可8:6-7；路22:18-19；林前11:24），其实真正使圣餐成为圣餐的关键不是祝谢，也不是祝福，而是领圣餐的人是否按理领用主的饼和杯，而不是主领或他人（林前11:27-29）。

8、圣餐里的饼必须是无酵饼吗？最好是。因为第一个逾越节用的就是无酵饼（出12:15）。但是为什么没有说必须是无酵饼呢？因为最初的逾越节是用无酵饼，不等于以后的逾越节都在吃无酵饼，圣经只规定了无酵节7天只能吃无酵饼，并没有规定逾越节只能吃无酵饼（利23:5-6；民28:16-17）。而且圣经中谈到圣餐的经文中的“饼”的原文是指犹太人日常吃的饼，可能是指有酵饼，而且从上下文来看并没有清楚说明是无酵饼（太26:26-30；可14:22-26；路22:15-20；约13:21-20；林前14:23-25）。有酵饼的原文是另外一个词，在林前5:7-8用了，所以有人认为如果主要求我们必须用无酵饼，就会清楚写明。因此我们认为饼和杯的象征意义是更重要的，而不是饼本身。虽然出于对第一个逾越节的尊重，我们尽量用无酵饼，但是特殊情况下用有酵饼也不是犯罪。圣餐到底用葡萄汁和葡萄酒的问题也是类似的。想阅读更多资料可以点击《圣餐用无酵饼还是有酵饼、用葡萄汁还是葡萄酒？》。

9、圣餐里的杯必须是葡萄酒吗？不一定。葡萄汁在当时是常见的饮品，耶稣也常常用葡萄做比喻（太20:1；21:33；26:29；可12:1；路13:6；约15:1）。圣经谈到圣餐的时候，原文的意思仅仅是喝“葡萄树的果实”，并没有明指是葡萄汁还是葡萄酒。其实葡萄汁和葡萄酒都属于葡萄树的果实。实际上葡萄汁放久一点发酵后就成葡萄酒了。在林前11:20-21提到当时哥林多教会吃主餐的时候会有人喝醉，所以当时圣餐用葡萄酒的可能性还是很大的。其实葡萄汁放久一点经过发酵，就会产生酒精了。现代葡萄酒的酿制只不过加入酵母来帮助发酵的过程而已。不过在我们圣餐的实践中，发现有的人对酒精过敏或不愿意粘酒精，所以为了爱弟兄姐妹的缘故，圣餐的时候我们通常使用葡萄汁。

10、圣餐的杯是必须所有人饮用同一个杯吗？不一定。圣经仅仅提到耶稣拿起一只杯来祝谢，但是并没有说所有门徒都饮用这只杯。所以有可能所有人都饮用这只杯，也可能耶稣把这只杯的葡萄汁分给其他人，然后每一个人喝自己的杯（路22:17），但是也可能耶稣只是用自己的杯祝谢，然后门徒各人喝自己的杯。从哥林多教会的实践来看，有人竟然会喝醉（林前11:21），所以各人喝自己的杯也是有可能的。

11、圣餐必须是先吃饼，然后再喝杯吗？不一定。从经文的描述，耶稣是先和门徒吃饼，然后才喝杯的。但是这个顺序并没有特别的含义或象征意义。也许是当时吃饭的习惯。而饼代表耶稣的身体，杯代表耶稣的宝血，这两者的意思都指向十字架，本质上代表的是耶稣的死（林前12:26）。所以有的教会在领圣餐的时候，会先领饼，吃完饼之后再领杯，有的教会则领饼之后再领杯，然后同时吃饼和喝杯。这都是可以的。

12、林前11:28提到的“自己省察”指指省察自己的罪吗？不是的。从上下文27-29节可以看到，我们必须按照圣餐的意义、正确的态度来领用圣餐，必须相信福音，如果不分辨是主的身体、主的宝血，就是吃喝自己的罪了，因为我们的罪只有主的身体替我们担当、主的宝血替我们流出才能得到赦免。所以这里的“省察”指是省察自己是否按照圣餐的意义、给与当有的态度来领用圣餐。详细的内容参见上面关于圣餐的意义和态度的解释。

13、有人问：前面讲圣餐用的饼最好是无酵饼，但没有说是必须，这是什么意思呢？这属于解经的榜样性原则，我们可以根据圣经讲的案例设立榜样，但是不是必须这么做。比如圣经并没有规定敬拜必须在周日，但是很多地方都记载了在周日聚会，所以我们跟随圣经的榜样来在主日聚会，但是如果在特殊情况下在其他时间敬拜也不是犯罪。又比如圣餐的杯很可能是葡萄酒，是有一定的酒精的，不然哥林多教会的人就不会喝醉了（林前11:20-21）。所以有的教会一直用葡萄酒。我们也曾经用过葡萄酒，但是我们这么长时间以来，发现常常有人对酒精过敏或不能喝酒，所以我们就改为用葡萄汁了，这也不是犯罪。而且林前11:23-29在提到圣餐的时候，23节提到了主命令我们要领圣餐，然后提到了饼和杯，说明圣餐必须有饼和杯，但是却没有对此做详细的规定，也就是必须用什么饼，必须用什么杯。其中最重要的是圣餐的意义和我们应有的态度。所以虽然逾越节的时候用的是无酵饼，耶稣和门徒用最后的晚餐的时候也很可能是无酵饼，根据圣经的榜样性原则，我们也最好用无酵饼，但是特殊情况下不能用或没有无酵饼，也不是犯罪。

二、从历史的角度

圣餐到底应该在哪里举行？以什么样的频率举行？我们一起来看圣经的历史，还有我们教会的历史。

圣经的第一个教会在耶路撒冷诞生，因为都是犹太人，他们一方面去圣殿参与犹太人的崇拜，一方面在家里搞家庭聚会。他们当时是天天聚会，天天在家里掰饼。所以很可能是天天领圣餐（徒2:41-46）。

当福音传到外邦，聚会的时间通常在七日的头一日，就是主日，掰饼也在主日，所以可能圣餐和崇拜在同一天（徒20:7）。直到今天，大多教会也在主日崇拜的时候设立圣餐。

在传统上教会设立圣餐的时候主要有两种方式：一种是在主日敬拜的时候设立，任何自认为信而受洗的人都可以领圣餐，包括外教会的人和本教会的人。有的教会在圣餐的时候会对不适合领圣餐的人做一些提醒，有的教会也可能会提前对一些不熟悉的人进行排查，但是基本上圣餐都是主日崇拜中的一个环节。另一种是单独有一个圣餐的聚会，有的教会对参加圣餐聚会的人是有要求的，比如是教会的成员或会友，有的则没有要求，任何人都可以参加，但是会宣告只有信而受洗的人才能领圣餐。关于这一点，圣经并没有明确的规定。

至于圣餐的频率，不同的教会会多种多样，有的每一个主日都领，有的一个月领一次，有的一个季度甚至更长。关于这一点，圣经并没有明确的规定。

至于圣餐的进行是在哪一个主日，不同的教会也不一样，有的放在第一周，有的在第三周，等等等等。关于这一点，圣经并没有明确的规定。

那么思麦教会的圣餐历史又是怎么样的呢？

我们一开始是每周都领圣餐的，有点模仿历史上第一个教会的模式，后来兄弟姐妹反映已经流于形式，希望频率降低一点，所以改为每一个月一次，一直延用到如今。

我们设立圣餐一开始也是在主日崇拜过程中进行，是其中的一个环节，后来等人多混杂之后，发现存在很多问题，比如没有信的人也领圣餐，受过洗但没有得救的人也领圣餐，异端邪教的人也可能领圣餐，还有领圣餐的时候一些不领的人已经开始退场，场面变得很混乱。同时我们教会施行了会友制，我们对于会友的定义是被教会认可是基督徒的个体。当一个人被开除会籍的时候，就等于不被我们认为是基督徒或被我们撤回对对方是基督徒的认可。详情请点击阅读《为什么要施行会友制？》、《为什么要加入教会》和《什么是健康教会的七标志？》。

所以为了更严肃地对待圣餐，也为了避免圣餐的混乱，也因为我们认为会友才是被认可的基督徒，我们后来经会友大会决议，把圣餐改为在针对会友的聚会里面举行，一直延用到今天。在会友的聚会里我们不仅仅有圣餐，还会重申我们的会友信仰告白和会友盟约，也会让会友们彼此相交彼此祷告，也会针对教会的重大事宜进行讨论和决议。当然这并不意味着以后永远不会改变。如果有两个以上的会友提出，经教师和长老同意，还是可以重新投票的。

三、从管理的角度

前面提到设立圣餐的主要两种方式，没有一种方式是完美的，圣经也没有就此做明确的规定。我们经历过第一种带来的问题，所以改为第二种。其实第二种也有第二种带来的问题。下面我们就这些问题来做一些解释或澄清。

1、什么样的人才能在思麦教会领圣餐？

答：目前只能是思麦教会的会友。也就是被思麦教会正式确认的基督徒，是思麦教会自己的兄弟姐妹。

2、别的教会的会友可以在思麦教会领圣餐吗？

答：非常抱歉，不可以。因为我们的圣餐是会友活动的一部分，一家人在一起除了领圣餐，还有其他重大事宜，本教会以外的人不适合在现场。

3、仅限会友才能领圣餐是否太狭隘，是否违背大公教会的精神？毕竟不是同一教会的基督徒也是大公教会或普世教会的一部分，也是基督普世的身体的一部分。

答：基督徒确实是大公教会或普世教会的一部分，也是耶稣基督普世的身体的一部分（弗1:23），与此同时，基督徒也应该是本地教会，耶稣基督在本地的身体的一部分（林前12:26-27）。圣餐就是为本地教会所设立的，而不是为看不见的普世教会设立的。地方教会在一起聚集的时候才进行圣餐的仪式，用来纪念主耶稣（林前11:20-26），代表他们同属一个本地的身体（林前10:17），所以同领一个饼和杯。所以不同的教会关于什么人可以领圣餐有不同的规定是正常的，这并没有违背大公教会的精神。

4、已经决定委身在思麦教会的人能领圣餐吗？

答：需要走完加入思麦教会的程序，被会友大会投票通过的人才能参加会友大会，才能领圣餐。所以一旦您决定委身思麦教会，下一步的就是尽快参加思麦教会的有关课程，经过面谈和会友大会的通过之后就可以成为思麦教会的会友，这时候就可以领圣餐了。对于某些人来说，这个过程可能显得有些长。求主赐给我们忍耐等候的心。详情请阅读《如何加入思麦教会？》。

5、如果我认为自己已经得救，却因为还不是会友而不能在思麦教会领圣餐，我是不是在犯罪？

答：不是的。因为这不是你的过错。你可能自认为是基督徒，但是还没有得到思麦教会的确认，所以需要等待是正常的。圣经中有个别的命令是需要有条件才能执行的。比如浸礼需要得到教会的认可，不是你自己在家就给自己洗的，圣餐也是如此。如果你客观上没有条件，比如和耶稣一起钉十字架的那位信主的强盗，就没有机会受洗，或者你还没有得到教会的许可，导致你不能参与主赐给教会的两大圣礼--浸礼和圣餐，你是不需要为此负责的，这是教会的责任，主也不会追究你的责任。

6、我已经信而受洗，但是思麦教会却因为我不是会友而不让我领圣餐，思麦教会是不是在犯罪？

答：既然思麦教会在会友大会做了这样的决定，就说明思麦教会有这样的定见（罗14：5），也有信心觉得是这么做并没有违背神的心意（罗14:22-23）。相反，神透过保罗在林前11:20-34责备当时的哥林多教会，因为他们在圣餐方面犯了罪，没有以正确的态度对待圣餐，以致于经历了神的管教，有患病的，死的也不少（林前11:30）。如果您有信心觉得思麦教会是在犯罪，请你为思麦教会好好祷告，求神开启思麦教会回归正轨。非常感谢！

7、我已经习惯了上面提到的第一种领圣餐的方式，就是在主日礼拜的时候任何信而受洗的人都可以领圣餐，而到了思麦教会却发现不同的传统，让我很不舒服，或很不能接受，怎么办？

答：我们能理解您的不习惯，每一个教会都有各自的传统，换到任何一个教会都需要有新的适应，这正是我们的生命上一个台阶的机会。弗4:2-3说我们要学会谦卑、温柔、忍耐、用爱心互相宽容，用和平彼此联络，竭力保守圣灵所赐合而为一的心。当我们在福音里面生命不断成长，我们的感觉就会改变，感恩的心就更增长了。

8、虽然我还不是思麦教会的会友，可是我非常渴慕领圣餐，特别想借着圣餐与主亲近以及纪念主，怎么办？

答：您的心情是可以理解的，非常为您的渴慕感恩，也为你暂时不能领用圣餐而感到难过。主已经悦纳了你的心，也必纪念你爱主的心。有好的渴望不等于一定要成为现实。在哥林多教会里，虽然很多人都得到主的启示，也渴慕分享出来造就大家，但是教会规定只能有2-3个人分享，其他的人就需要学会闭嘴（林前14:29-31）。同时也请您理解我们关于圣餐的规定。教会需要规规矩矩地按着次序行（林前14:40）。既然思麦教会做了这个信心的决定，就需要执行，否则就是不信实，以后任何人都可以以类似的或其他理由打破规矩，这样的话规矩的存在就失去了意义。亲近主的方式有很多，圣餐只是其中一个。所以请您多多祷告，您顺服教会的决定的心也是主所喜悦的，您为此付出的忍耐也是主所喜悦的。求主亲自安慰你的心并因你爱主的心而大大赏赐你！

9、如果我不能领圣餐，是否失去了领圣餐所带来的祝福？

答：领圣餐是主所颁布给每一个基督徒的命令（路22:19），顺服主的命令肯定会带来祝福（雅1:25）。但是主也说要顺服教会和教会的领袖（太18:17；来13:17）。所以因为主的缘故，顺服教会和教会的领袖也会带来祝福。同时，如果不是你个人的原因，而是教会的原因导致你暂时不能领圣餐，这并不是你的责任，你并没有得罪主，所以你并没有失去领圣餐而带来的祝福，你得到的是顺服教会和教会的领袖而带来的祝福。所以您不必担心。凡事都有主的美意。愿主因着你的顺服大大祝福你！
