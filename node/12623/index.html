# 单身的误区之四

　　误区四：“我的要求并不高，我已经放下了很多其他的择偶条件，唯一的要求是对方在面对挑战时能与我同担责任，是一个负责任的男人，而不是撂挑子，这样就行了”。或者“我的要求已经最低了，只要爱主就可以了”。
 
　　当谈到择偶的时候，我们通常会考虑自己有什么样的择偶条件，这些条件无非是三种：一种是客观的条件，比如房子、财产、收入、地域、教育程度、年龄。。。这些都是硬指标，是客观可以衡量的，比如说要嫁比自己年龄大的，这是很清楚的标准。一种是主观的外在条件，比如相貌、口才、厨艺、做家务能力、工作能力。。之所以是主观，是因为这些条件难以有统一的衡量标准，完全靠个人的评估。比如什么样的长相才是英俊或漂亮？不同时代的不同人可能有不同标准。一种是主观的内在条件，比如品格、气质、信心、灵性。。。这些也难以有统一标准衡量。比如说是否常去教会就肯定爱主、有信心？其实不一定。
 
　　因为客观的条件容易提，所以当你问一个人的择偶条件的时候，人很容易想到，也很容易说出来。主观的条件就不好说了。但是考虑周全的人也会以某种方式提出来。比如说希望对方温柔、幽默、大方等等。或者希望对方爱主、能一起服侍主等等。
 
　　客观的条件虽然我们很容易提，其实这些并不是择偶成功的关键。因为婚姻毕竟不是做买卖，虽然我们口头上会提一些外在的条件，主要是经济条件、教育条件，其实在真正见面的时候还会看客观条件以外的主观条件。当我们细细扪心自问的时候，我们可以知道：我们嘴巴说出来的条件其实都不是我们所有的条件。当我们看重外在的客观条件的时候，其实我们并不知道自己真正想要的，也不知道婚姻中真正重要的是什么。很多人找到了外在合乎自己要求的人，很快又大失所望。因为发现这并不能给自己带来真正的幸福。
 
　　 根据调查，过于理智，过于强调经济收入、长相、住房等外在条件，或者把这些条件摆在第一位，而没有摆正情感和爱情在择偶中的位置，是大龄青年择偶面临困难的主要原因之一。
 
　　当谈到择偶条件或标准时，我们首先要问一个问题：提择偶条件有圣经依据么？我们先来看亚当的例子，创2:18-23节说耶和华神给亚当早一个配偶之前只是让亚当睡了一觉，然后神就把为亚当造的配偶带到了亚当的面前。神有没有问亚当要什么样的配偶？没有。有人说神没有问的原因是因为亚当没有见过女人。或者没有其他女人可以比较。这种说法的错误在于我们把我们现在的情景读入了圣经的经文当中，我们把自己假设成为亚当，而且认为自己在没有见过其他女人的情况下不会提出也无法提出什么要求来，所以神也不需浪费功夫，直接给亚当造一个就好了。但是其实我们并不是亚当，也无法完全体会当时亚当的情景，也无从了解亚当当时内心的审美标准。而且亚当看见过那么多美丽的飞鸟走兽，也有自我的认识，包括对自己身体的外表的认识，他可以跟神说：我要一个跟我模样差不多的人--也就是男人来帮助我。亚当也可以跟神说：我希望要一个有老鹰类似的眼睛的人（老鹰的眼睛非常锐利），或者要一个有猴子类似的四肢的人（猴子爬树很厉害）。。。亚当可以提出很多自己认为不错的理由，但是他没有提。神也没有问他，我们需要注意，亚当这时候还没有犯罪，也就意味着他完全相信神，完全信任神，他安心地睡着了，他相信神做事是对的，神的道路是最好的，他安心等候，等候神赐给他完美的配偶的时刻，因为他已经发现自己的需要--需要一个配偶做适合的助手（配偶的原文就是“适合的助手”的意思），而他在自己管理的动物世界中没有找到自己的配偶。他相信神的预备，所以沉睡了。当神把夏娃带到他面前的时候，他知道夏娃是从他身上取出来的。他怎么知道的？我相信是神告诉了亚当。所以亚当确认眼前这个人是自己骨中的骨和肉中的肉，因为是从他身上取出来的，所以亚当在神面前和夏娃立了盟约，成为地球上第一对夫妻。所以我们看到亚当在择偶方面并没有跟谁提任何的条件，当他明白神的心意后就选择了顺服。
 
　　然后我们再看亚伯拉罕为自己的儿子以撒娶妻子的例子。亚伯拉罕把这件事交给他的老仆人去做，显然老仆人是敬畏神和神亲近的人，从他帮自己主人找儿媳妇的过程就可以看出来。当老仆人到拿鹤的城的时候，他跟神提出要确认神所预定做以撒的妻子的两个条件：1）给自己水喝。2）给自己骆驼水喝。当遇见利百加满足这两个条件之后，老仆人继续寻求，他问利百加的父亲家是否有住宿的地方，结果有；到了利百加家中，老仆人又问是否同意利百加嫁给自己主人的儿子，结果是同意；最后再问利百加是否同意马上和自己回去同以撒成婚，结果也成就了（创24章）。每一步这位老仆人都依靠神和敬拜神。他十分确认神透过对他的祷告的带领，包括对他所提的为以撒择偶的条件的应允。从创24:14可以看出老仆人对自己提出的择偶条件的信心，他等于说：如果这件事发生，就是神预备给我主人儿子的妻子了。
 
　　所以当你向神和人提出择偶条件的时候，需要问自己一个问题：1）这样的条件客观么？可衡量么？2）我有信心接受符合这样条件的人么？也就是说一旦符合条件，就要结婚。而不是还要通过恋爱阶段作为过渡，如同以撒和利百加。3）我相信这样的条件是来自于神或合神的心意么？因为这样的条件如同印证。如果你真的有来自神的信心，可以效法老仆人如此向神祈求，然后步步寻求神的带领。如果你没有来自神的信心，你需要再问自己一个问题：既然我原来的这个择偶条件不一定是神所预定的，那么我为什么要确立这样的条件呢？这样的条件是从哪里来的呢？
 
　　一、和神的话语相对照，神的话语可以照明我们心中的意念并帮助我们辨别。来4：12-13“神的道是活泼的，是冇功效的，比一切两刃的剑更快，甚至魂与灵，骨节与骨髓，都能刺入、剖幵，连心中的思念和主意都能辨明。并且被造的没冇一样在他面前不显然的；原来万物在那与我们冇关係的主眼前，都是赤露敞幵的。” 
　　二、祷告求主显明。如同诗篇139:23-24说“神啊！求你鉴察我，知道我的心思；试验我，知道我的意念。看看我里面有什么恶行没有，引导我走永恒的道路。” 我们需要神来帮助我们明白自己的内心，因为人的心比万物都诡诈，只有神才能识透。
　　三、因着主的怜悯，当神光照你看到自己择偶条件背后错误的假设、错误的动机、错误的信念等等的时候，你所要做的就就是把这些抛弃，用圣经的真理来替代。
　　我们可能会发现我们所谓的择偶条件可能来自几方面的影响：
　　1）你的自私或罪性。潜意识里，我们都想找完美的人，找一个各方面都让自己满意的人，从外表到内在，可能因为自己觉得自己条件不好，所以降低一点择偶标准。我们择偶主要考虑的是满足自己的期待和欲望。或者满足自己父母的期待和欲望。我们总是容易从自己的需要出发。当两个人了解越多，你会发现自己还有更多的需要没有被满足，甚至原来以为能被满足的需要对方都没有做到，结果你就想放弃了。我们需要省察自己：我的哪些条件主要是为了自己?
　　2）你的父母。你对异性最早的认识多半来自自己的异性父母。如果你是女孩，父亲身上你认为美好的东西，或者你的父亲所认为一个男人所应有的美好的东西，常常是你希望在你的配偶身上看到的。你父亲身上你所不喜欢的东西，或者你父亲所教导你一个男人不应该有的东西，也是你希望你未来的配偶身上所没有的。我们为什么会受这样的影响？我们可能会有以下假设：我父亲那样的男人是最适合我的或对我最好的，或者我父亲的看法一定是对的，因为他爱我，他也是过来人，见多识广，或者他的话语带着权柄。不过我们要问自己一下：这些假设有圣经依据么？或者一定符合神的旨意么？或者真的符合真相么？结果你会发现，那是不一定的，即便你的父亲是一个在你眼中近乎完美的父亲，也最多说明他最适合你的母亲，但是不等于适合你。另外，如果婚姻只是为了找一个对自己好、给自己安全感、适合自己的人，这肯定是神对于我们婚姻的计划么？你会发现，如果婚姻成为你自我为中心的一个产物，你一定会遭受挫败。而且你也可能发现，在现实生活中你可能喜欢上的偏偏是你父亲不喜欢的那种类型，或者有你爸爸的一些特别的缺点的男生，或者和你爸爸的优点相反的男生，这时候你又会怎么办呢？
　　3）社会教育。教育无处不在，透过学校、媒体。。整个社会在告诉你应该找什么样的人，而且你会发现不同朝代的标准还不一定一样（古代女人以丰满为美，现在以苗条为美），不同媒体的看法也可能不同，或者你不同的朋友会给你不同的建议。也包括你的牧师或教会给你的建议或教导。你需要问自己：我为什么认定这样的人适合做我的配偶呢？是因为大家都这么认为？是以为这是这个时代流行的？是因为周围人都这样选择？。。。这样的标准符合圣经么？或者有圣经依据或支持么？
　　4）过去的经历。如果你曾经因为自己贫穷被曾经一个女孩抛弃，你可能会因此相信女孩都很看重钱。你因为要找一个对钱一点都不在意的人。这样在思想或行为表现上有一点看重钱的女孩你都可能会呲之以鼻。如果你过去被人欺骗过，也许让你很难相信别人。如果你过去的男友花心或丈夫有外遇，你可能会觉得男人都是这样，如同天下乌鸦一般黑。我们过去很多负面的经历都容易给我们带来一些谎言，让我们相信一些扭曲的东西。如果不通过神的话语和圣灵的光照，我们容把他们当真理看了，结果害人害己。
　　5）宗教教育。包括你从各种基督教媒体得到的信息、从教会得到的信息等等。如果牧师告诉你配偶应该找爱主的，你就会如此行。那么什么才是爱主的呢？如果牧师告诉你每周来敬拜的、每周参加小组的、热心来服侍的就是爱主的，你也会这么认为。
 
　　当然可能还会有其他途径的影响。总之我们需要主帮助我们甄别：这些想法和条件到底来自于哪里？动机是什么？是否真的符合圣经和神的心意？
 
　　如果因着主的怜悯，他光照你看到自己择偶条件背后错误的假设、错误的动机、错误的信念等等的时候，你所要做的就就是把这些抛弃，用圣经的真理来替代。因为神的旨意才是纯全、善良、可喜悦的旨意。
 
　　另外择偶方面还容易存在的一个问题是大家都觉得自己的择偶条件不高，尤其是大龄青年们，或者有很多恋爱或婚姻经历的人，他们认为自己已经放下了很多条件，目前剩下的已经是底线了，不能再没有原则了。其实恰恰相反，如果一个人的择偶条件不是来自于神，哪怕只有一条也是高的（圣经说律法本是让人知罪，你定下一条绝对的标准，就等于给对方颁布一条律法，只会让你更多机会定别人的罪），而且年龄越长的人越觉得自己冒不起这个险，所以会抱着自以为底线的条件在死扛。对于基督徒来说，提出的条件通常都比较“属灵”，表面上看起来合情合理，其实高不可攀。比如一个女生说要找一位爱主的男生，请问什么叫爱主？有什么具体的标准来衡量？通常是以这位女生心中的标准来衡量，而这样的标准可能会不断攀升，或包含越来越多的生活层面。当你有几次不去教会的时候，她可能会认为你不爱主，当你约会迟到的时候，她也会认为你不爱主，当你在教会服侍不如她想象的那么积极的时候，他也认为你不爱主，当你有时候被罪所胜对他发火的时候，那就更糟，她不仅认为你不爱主，而且认为你连她也不爱了。有女生想找能和自己一起服侍主的男生。她的意思通常是和自己在某项事工或教会里面服侍。不过我们常常容易忘记婚姻是我们最大最重的服侍工厂。当你进入婚姻之后，你的配偶是你最需要服侍、也是最难服侍的对象。你是愿意更好地服侍你的配偶来服侍主呢？还是想你的配偶和你一定做一些你想做的事情来服侍自己的愿望或想法？在婚姻中的人都知道：服侍别人通常是比服侍配偶容易的。也有很多女生想找负责任的男生，至少能一起面对困难或挑战的男生，这也很难。因为男生遇到问题最容易选择逃避，这是男生容易犯的罪，而女性容易犯的错误就是一有问题就要男生必须给个说法，或者必须限时主动解决，或者必须以自己认为合理的方式解决。这样的压力会让男生更容易沉默或逃避。结果呢？女生就会给男生扣上一个帽子：不负责任。另外，当女生要找一个所谓负责任的男生时，通常是想找一副可以依靠的肩膀，让自己更有安全感。不过我们扪心自问：这个世界哪一个男人没有罪呢？有罪的人又有谁能成为你的靠山呢？圣经说也靠耶和华，胜似依赖王子。（诗篇118:8-9）那些以人为靠山的人注定是要失望的。根据调差，很多男人在订婚之后逃婚，就是后来发现女方所要自己承担的担子或责任让自己无法承受。
 
　　所以恳请大龄青年们想一想，我的条件虽然表面上很“属灵”，真的是来自于神的么？很多人因为这些“属灵”的择偶条件，最后只能嫁给耶稣。因为没有人能真正满足她们的条件。婚姻是一个盟约，等于决定一生一世不顾一切来保护和爱对方，如果一开始就从自己出发寻找对自己有利的条件，婚姻的挫败其实就已经注定了。
 
　　也有人说：“虽然大龄了，但是我不贱卖”。我想说的是：亲，婚姻不是买卖，除去自己那些扭曲的择偶条件也不是贱卖，而是为了荣耀主而放下自己。主会在婚姻中赐给你很多祝福，也会带来很多挑战，因为神赐给你婚姻最终的目的不是为了地上，而是为了天上，为了你享受天上永恒的祝福，为了你荣耀那位为你舍命的主，在天上就不存在婚姻了，我们最终的新郎是耶稣基督。完美的爱只有天上有。这就是为什么主可能会用地上的患难让我们更加期待未来。我们通常的错误是对地上有了过高的期待。如果你过去曾经被异性伤害或抛弃，鼓励你常常回到主的爱里，在他的爱里得到医治和恢复，包括除去过去留下的谎言，包括让你有宽恕和交托的心态等等。其实没有一个异性配得你委身一辈子，因为没有一个人值得我们一生去依靠。你之所以选择委身一个人，不是因为这个人完全满足你的条件或期待，不是因为这个人配得你如此委身，而是为了在婚姻中荣耀爱你的主，为了在婚姻中活出主赐给婚姻的意义，为了和这个人在婚姻中一起成长，为了在婚姻中彰显基督和教会的关系，为了预备你自己进入永恒。。。你进入婚姻的诸多目的当中为你自己的部分越少，你享受婚姻的可能性越大。我们常常犯的错误是舍本逐末，把自己的愿望、计划、期待等摆在前面，结果更容易挫败，神希望的是我们把神和别人摆在前面，这样我们会更加蒙福。雅各书1:25说顺服神的“在所行的事上必然得福”，说就是这个道理。所以大龄的你如果放下不该有的择偶条件，你并没有贱卖，而是更加显出你的高贵，因为你知道你是万王之王万主之主的儿女，你知道你如此地被爱，你知道你的婚姻最终不是为了自己，你也知道你不可能在婚姻及对方身上得到最大的满足，你也知道没有一个人配得你如此舍己地委身一辈子，但是那位爱你的为你舍命的主配得，配得你在婚姻里荣耀他。当别人看到，也当你自己感觉到：如果不是为了主，你不会如此爱这个异性，你不会如此对他委身，你甚至不会和他进入婚姻，当这些事情发生的时候，主在你身上就得更多更大的荣耀了。阿门！
 
　　有人说：“不是很多人给神列一个未来对象的条件的清单都成功了么？有人拿着这样的清单向神祷告，结果神就应允了。我们为什么不可以如此效法呢？”是的，的确有这样的成功案例。这能说这样的婚姻是神的恩典和怜悯。但是不等于这是对所有人都适用的，或神一定听所有的类似的祷告，或这样做一定是神所喜悦或荣耀神的。旷野中的以色列人因为厌烦了吃吗哪，想要吃肉，神让很多鹌鹑飞来，结果怎么样呢？（民11章）神听了以色列人的请求给他们一个王，名叫扫罗，原因是以色列人不想让耶和华做他们的王，结果怎么样呢？所以最重要的是要问自己：我为什么要列这些清单？我为什么要神一定按照这个清单为我造一个配偶？
 
　　最后，圣经里是否有基督徒的基本择偶条件呢？有的。首先，按照创2：24，你的配偶是异性，而不是同性。很多人会说这是废话，不过对于当今的世界来说，还是有必要提一提的。然后你的配偶必须是基督徒。林前7:39说要嫁在主里的人。如果一个人真的是基督徒，他一定是爱主的，只是多少的问题。本来基督徒的基本择偶条件很简单，不过很多人把世俗的条件加上去，最后导致基督徒结婚的条件比世俗的还高，因为不仅仅要求世俗所要的东西，还加上了属灵的要求。所以有人叹息结婚难或找对象难的时候，需要问自己：是否自己的条件太高了？太“属灵了”？有人说基督徒结婚难的主要原因是基督徒不仅仅要非基督徒择偶所要的条件，而且还要加上属灵的条件。这话对很多人来说是真的。有人传道人希望自己的爱人和自己一起在某个方向上服侍，或者说属灵上有符合自己标准的成熟度，这都是很高的条件，等于是找同工，或战友，而不是配偶，结果会导致自己一直停留在剩男剩女的行列。本来基督徒进入婚姻是更容易的，但是因为很多误区反而变得更难了。
 
　　有人因此说，那么是否和任何一个异性的基督徒都可以结婚？
　　一是从圣经的角度确实你可以跟任何一个基督徒结婚，因为这是圣经讲的基本条件；
　　二是你生命中认识任何一个基督徒都不是偶然的，都有神的预备，你也不可能认识所有的基督徒的异性，你也不会对所有的基督徒异性产生感情，也不是所有基督徒的异性都愿意和你谈恋爱和结婚，这其中有神的客观和主观的带领。我们要相信婚姻是神的预备，是神主权的作为，我们只是寻求神的旨意。我们有理由相信，因着神的爱，最后肯定只有一个人和你走向红地毯，不是是很多人。：）
　　有人会担心：神会不会强迫我嫁一个我不喜欢或不愿意嫁的人？不会的。你想一想，你的天父爱你爱到为你舍弃他的独生子，他最喜悦的是你因为相信他、爱他而顺服。如果神非要你做一件事情，最终神会赐给你所需要的信心。神让何西阿娶一个淫妇，神也给了何西阿愿意顺服的心。婚姻是你一辈子最重要的决定之一，是你自己的选择，你最后进入婚姻的人那位一定是你愿意或选择要嫁的。这个人未必所有方面都让你满意，但是最终一定是你愿意委身一生的。圣经说寡妇可以“随意再嫁，只是要嫁在主里的人”（林前7:39）。何况对于你呢？所以如果神一定要你嫁给某一个人，他可以改变你的心思意念，让你愿意嫁给他，所以你绝对可以放心，不需要有任何担心惧怕。当被难以接受的潜在对象追求的时候，你可以省察自己：我为什么不能接受对方？是不是择偶标准有问题？或我的内心有问题？我有什么需要悔改的地方么？你也可以如此祷告：主啊，你是全能的，如果是你的心意，求你改变我的心。有人就这么成功了。神所要求于你的，他自己会供应。耶和华是供应和预备的神。我们想一想：在耶和华岂有难成的事么？神赐给我们的不是恐惧的心，是刚强、仁爱、谨守的心（提后1:7）。所以我们只需要刚强壮胆！
 
　　有人可能会问：如果我确实不确定神要给自己的是否是自己择偶条件里面的，是否就肯定不能按照自己的条件向神祷告？不是的，如果你去除不合主心意的条件，剩下的条件你都可以向主开口，不过你需要效法耶稣在客西马尼园祷告的榜样，耶稣把自己真实的想法告诉了天父：父啊，求你将这杯撤去。（可14:36）我们需要对上帝真实，等于对自己真实，因为神是无所不知的。所以你可以把你的任何想法告诉上帝。然后要效法耶稣的交托和顺服。耶稣说：然而不要成就我的意思，只要从你的意思。耶稣最后所求的是天父的旨意成就。如果我们真的信主，就要相信神给我们安排的，可能不是我们想要的，或者不是我想象的，但是神是因着爱带到我们生命中的。罗12：1-2告诉我们当我们越决志顺服神的时候，我们就越能明白神的旨意。而且神的旨意是纯洁完美的、良善的、可喜悦的旨意。我们之所以跟随主，是因为我们相信主对我们是良善的，主就是我们的道路、真理、生命。阿门！
