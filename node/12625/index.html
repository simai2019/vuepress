# 单身的误区之六

　　误区之六： 神为我预备的配偶只有一个，千万不能错过，所以一定要等我确认是否是神所预备的才能和对方谈朋友或结婚。 


　　这个说法好像在一些基督徒圈子里很流行。我们都知道神如此爱我们，如果他为我们预备的一定是最好的，既然是最好的，怎么能错过？而且婚姻是终身大事，一旦错过，后悔也来不及啊。而且有些姐妹也遇到过这样的事情：有的弟兄来到她们面前说，自己得到某种启示或感动，她就是神为他预备的，让这些姐妹感到怪异，尴尬，不舒服或无所适从。 
　　首先我们要明白“预备--prepare”的意思。出23:20 "看哪，我在你面前差派使者，好在路上保护你，领你到我预备的地方去。"这里所预备的地方是指神为以色列民预备的迦南美地，最后神成就了没有？成就了。太25:34“ 那时，王要对右边的说：‘蒙我父赐福的，来承受创世以来为你们预备好的国吧。 ”这里所预备好的国指的是主为他的子民预备的天国，最后神成就了没有？成就了。太25:41 “王也要对左边的说：‘你们这被咒诅的，离开我，到为魔鬼和他的使者所预备的永火里去吧！ ”这里所预备的永火就是指神为不信他的人、魔鬼和他的使者预备的地狱或火湖。最后神成就了没有？成就了。林前2:9“正如经上所记：“　神为爱他的人所预备的，是眼睛未曾见过，耳朵未曾听过，人心也未曾想到的。” ”这里所预备的是指十字架的救恩，也就是耶稣基督并他钉十字架。这神显然已经成就了。所以当我们说神预备做某事的时候，是一定会成就了。否则神就失败了。神是绝对不会失败的，他命有就有，命立就立，没有人能拦阻他的旨意。神是凭己意行万事的神。当这个真理应用在婚姻中的时候，如果说某某是神所预备给某人的，就等于他们一定会进入婚姻，因为神所预备的绝对不会失败。怎么知道谁是神为你所预备的，你结婚的时候就知道了。


 　　因此如果我们想知道谁是神所预备的做我们的配偶，结婚的时候就知道了，婚前知道这个对我们来说没有实际意义，因为那是一定会发生的事情，是我们无法左右或拦阻或促成的。而实际上很多兄弟姐妹做这个祷告的时候，也就是求神告诉自己谁是神所预备的，他们的意思是想知道神是否希望他们和对方进入恋爱关系或婚姻，这等于寻求神的心意，然后他们会顺服。（有意思的是，很多人表面上在寻求主的心意，其实自己心中已经有了偏好，如果得到的答案和自己心中所想的不一致，就会怀疑是否是神的心意。这等于是在试探神。也有很多人在一些重大事情上着急寻求神的心意，其动机不是为了顺服神，荣耀神，而是想因此得到某种保证或安全感，觉得神让我嫁娶的一定会让我满意或给我幸福。我们似乎忘记了耶稣是因为顺服神而上十字架的。）
 
　　接下来的一个问题是：神是否一定会告诉你他希望你和谁恋爱或结婚？当然有些人是神肯定不希望你和对方结婚的，也就是神对于婚姻的最低条件，也就是对方必须是异性和基督徒，这个问题在前面的误区讨论过。这也是“如何寻求神的旨意”中的一大内容。很多兄弟及在这些方面接受了不完全或错误的教导，所以变得非常困惑。这个问题从更大的范围来说，就是：神会在所有事情上告诉你应该怎么选择么？
 
　　一、圣经已经告诉我们关于生命和信仰的所有原则上的心意。彼后1:3 "神以他神圣的能力，因着我们确实认识那位用自己的荣耀和美善呼召我们的，把一切有关生命和敬虔的事，都赐给了我们。"提后3:16-17“圣经都是神所默示的，于教训、督责、使人归正、教导人学义都是冇益的，叫属神的人得以完全，预备行各样的善事”。这里告诉我们圣经已经足够我们活出神喜悦的、合神心意的生活。也就是说一般情况下圣经里面的原则、命令、智慧等已经足够我们做生活中的决定。圣经说“你的话是我脚前的灯，路上的光”。而我们常见的错误是：我们抛开圣经显明的主的原则和心意不谈，非要透过某种特别的方式来追求主的心意，结果越来越困惑。比如说关于今天是吃白菜还是萝卜，而和你同住的爱人喜欢吃萝卜，也许因着爱的原则，在只能做一种菜的前提下你应该舍己买萝卜给爱人吃，而不是先考虑自己。圣经有这样的应许在罗12:1-2节“所以弟兄们，我以神的慈悲劝你们，将身体献上，当作活祭，是圣洁的，是神所喜悦的；你们如此事奉乃是理所当然的。不要效法这个世界，只要心意更新而变化，叫你们察验何为神的善良、纯全、可喜悦的旨意。”也就是说我们首先要在想法上和行为上脱离这个世界的败坏，不断根据圣经来矫正，活出圣经的样式，然后我们就会越来越明白神的旨意。这是何等宝贵的应许！另外圣经也有很多智慧来帮助我们做正确的决定，不过基督徒当中有多少认真查考圣经，明白主对我们的心意呢？弗5:17说的好：不要做糊涂人，要晓得主的旨意如何。
 
　　二、圣经告诉我们神可能会以某种特殊方式带领我们做具体的事情，这是少数的情况。拿伟大使徒保罗的例子来说，圣灵主动带领他，或主亲自像他显现指示他做具体的事情，这些情况在圣经里记载的并不多。包括主在大马士革路上向保罗显现，赐给保罗一生的使命（徒22:10-16），后来主再次向保罗显现让他离开耶路撒冷去外邦人传福音（徒22:17-21），圣灵差遣保罗和巴拿巴开始第一次宣教之旅（徒13:2），主透过异象带领保罗去马其顿传福音（徒16:9），主派使者告诉保罗他和整船的人一定会脱离患难（徒27:23-24）。等等。其余大多数情况下保罗根据圣经的原则以及自己的愿望、理性、负担等来做决定。比如保罗被神的话语激励下定心志在没有传过福音的地方去传福音（罗15:20-21）。保罗通常去犹太人的会堂传福音，因为那里聚集很多犹太人（徒13:5，14，44）。保罗安排别人和他一起去给耶路撒冷送赈灾的款项，因为免得别人怀疑自己在钱财上不洁，以让自己的善行被人毁谤（林后8：16-21）。如果我们继续查考新约中关于主超自然地介入信徒的生活中带领他们的例子，我们可以得出如下结论：1）这些例子是很罕见的。都是发生在非常特殊的情况下，主把特殊的心意告诉他所爱的门徒。2）这些例子都是神采取主动介入，不是人靠着祷告求来的。这并不是说祷告不会帮助我们明白神的旨意，而是强调神的主动性和主权，如果神一定要你明白他在具体事情上的心意，神会有办法让你明白的，因为在人不能，在主凡事都能。主也可以对你的寻求保持沉默，让你按照圣经的原则自己来做决定。当然如果神真的给你这样的信心，也就是说你通过祷告等方式一定能寻求到神在某件具体事情上的心意，你就要忍耐等候和不断寻求，直到主向你显明他自己的心意。
 
　　亚伯拉罕的老仆人是一个寻求神的心意的美好例子：他为以撒选妻子的时候曾祷告神说：我对哪一个少女说：‘请你放下水瓶来，让我喝点水。’如果她回答：‘请喝，我也给你的骆驼喝。’愿那少女就作你选定给你仆人以撒的妻子。这样，我就知道你施慈爱给我的主人了。” （创24:14）这位老仆人实际上是在说：如果有一个女孩符合我所提出的条件，就求你指定这个女孩做以撒的妻子，换句话说让这个女孩做以撒的妻子就是你的心意。之后虽然神的心意向这位老仆人显明，但是不等于利百加和他的家人同意，所以老仆人继续一步一步寻求神，祷告神，每当他看到神给予通达的道路的时候他都立马敬拜感谢神，直到最后利百加跟随他回到以撒的家中。
 
　　有些人热衷于依靠所谓的圣灵的感动或脑海中出现的话语来确定神的心意，然而他们所得到的信息却未必和圣经中神的话语相一致，很多仅仅是自己的冲动或感觉，结果给别人带来很多伤害和疑惑。我们要记住，圣灵的工作是荣耀基督，是要把基督的心意启示给我们，是不会和神的话语相矛盾的，因为神是一致的神。无论我们从什么手段或途径接收到任何信息，都要透过认真查考圣经来验证。因为你自己或撒旦也会让不同的信息出现在你心里或脑海里。
 
　　三、神只要你对他显明的心意负责。当神透过某种方式告诉你他希望你做什么的时候，你需要为此负责，你必须顺服，且竭尽全力做好，因为你要为此向主交账。但是当主没有告诉你具体事情上的心意的时候，你就不用对此负责，你只要根据圣经的充足的原则来做决定就好。这给我们很多释放，不是么？（当然也给我们很多压力，就是我们必须认真查考圣经）。如果我们总是以为什么事主都会具体指示我们，我们必须先明白主的指示才能走下一步，我们会疯掉的。因为我们大多数情况下没有得到主的具体指示，不知道主的具体心意如何，如果因此我们一点也不敢动，我们会有无尽的担心，因为我们也不知道主是否希望我们一直按兵不动。比如今天吃白菜还是萝卜，你可以祷告等候1天，还是没有主的指示，你不知道该不该吃，该吃什么，也许主早允许你吃猪肉了，你还祷告白菜和萝卜的问题。或者主根本不希望你在这件事上需求和等候，你直接买任何你爱吃的就可以了。当我当父亲之后，更明白这样的道理：我对于自己的儿子是有道德上的要求的，有时候也会告诉儿子我希望他做什么，在我不主动介入儿子的生活的时候，我希望他按照我给他的原则做决定就好，他有很多的自由，他可以享受这个家庭给他的祝福，因为他是我的儿子。当儿子还征求我对于某件事情的意见的时候，我可能会直接告诉他我的意思，也有可能让他按照以前的原则做就好，也有可能让他自己做任何自己喜欢的决定，因为这件事情没有原则性问题。
 
　　我们还记得神在旷野引导以色列人的经历么？出13：:2-22说白天神用云柱，晚上神用火柱来引导以色列人的道路。申1:33告诉我们神一直在以色列民的前面引导他们。民9:15-22告诉我们什么时候云彩从帐幕上离开，以色列民就前行，什么云彩停住，以色列民就安营扎寨，云彩停留多久，以色列民就在那个地方呆多久。有时一两天，有是一两个月，有时甚至一年。是否云彩停留的时候神就不带领以色列民的生活了呢？不是的，当以色列民停留的时候，他们需要神给他们的律法来生活，那也是神给他们所定的旨意。所以当神带领我们更换人生的环境的时候（可能神主动显明，也可以神给我们信心让我们透过祷告等方式来寻见），我们需要顺服，当神没有显明他的新的带领的时候，我们则需要根据神已经启示在圣经的原则做决定。
 
　　我们还记得所罗门和耶和华的经历么？所罗门被誉为最有智慧的人之一。他一生当中耶和华向他总共显现过几次？答案可能让我们很惊讶，一共才两次。第一次是在基遍，神在异梦中向所罗门显现，问他要什么，所罗门说要智慧，神就应许给他智慧（王上3:5）。第二次是在圣殿建造完成之后，神第二次向他显现，对他说：我已经应允你的祈求，如果你听我的话，我就坚固你的国位，如果你或你的子孙违背我，就要遭致管教（王上9:2）。平时所罗门怎么生活？他遵行神的律法（王上3:3）。他用神给的智慧判断事物（王上4:29）。然后他把神给他的智慧写在圣经里，就是传道书、箴言等里面所罗门的作品（王上4:32）。我们今天同样拥有神的话语，拥有神的智慧，而且我们祷告寻求神的智慧，然后做最好的决定（雅1:5-8）。最重要的是信心，我们信主的人心中都有圣灵，当我们相信神会赐给我们智慧，让我们按照圣经的原则做决定的时候，我们就会得到智慧。
 
　　以上的原则应用在婚姻就是：一般情况下你根据圣经的原则就可以做合神心意的决定，在不违背圣经的原则的前提上你可以做合乎自己心意的选择（当然我们要省察自己的动机和行为是否合乎主的心意）。林前7：39说“丈夫活着的时候，妻子是受约束的；丈夫若死了，她就可以自由地嫁给她愿意嫁的人，只是要嫁给主里的人。 ”这里我们可以看到寡妇是可以嫁给自己想嫁的人，这里保罗没有说：你先要明白具体谁是神给你预备的，然后你再嫁给他。所以在婚姻上你是有自主权的。当然也会有一些特别的情况，就是神给一些人特别的信心，让他们相信神一定会告诉他们谁是他们的恋爱对象或结婚对象，所以他们会忍耐等候和不断寻求。也许神透过神的话语、特别的印证等方式来让他们确认这些对象。这是有可能的。神如何在过去如何带领保罗和亚伯拉罕的仆人，也有可能在今天带领我们。各人凭着信心在神面前守着就好了。
 
　　所以我们确实地知道：神不一定会告诉你你应该和谁恋爱或结婚，神会根据圣经的原则来引导你，因为末日神会根据圣经来审判你。神也会透过环境来带领你，因为恋爱和婚姻是两个人的问题，不是你一个人就可以决定的。一般情况下（除非神特别启示你），不存在爱人的唯一性的问题，也不存在错过就不会再来的问题，你一生可能恋爱很多次，最后才结婚。这是很正常的，这并没有什么问题。真正有问题的是你在和异性交往过程中是否遵循了圣经的原则，包括你的动机、态度、方式等等。当然也不要轻易恋爱和轻易分手，因为这涉及爱的原则。恋爱是要负责任的，分手是容易受伤的。因为恋爱是为了往婚姻的方向走，离开爱情如同戒毒，深深陷入的一方是很难自拔的，而且失恋的痛苦可能在分开之后才真正开始品尝。林前16:14 “凡你们所做的事，都要凭爱心而作”。
 
　　因此当你爱上某个人的时候，如果没有超出圣经的原则和范围，你就可以带着负责任的态度和对方恋爱，当然一个巴掌拍不响，你可能要去追求，也可以允许被对方追求。当某个人爱上你的时候，如果没有超出圣经的范围，你也可以考虑是否用爱来回应对方，你需要谨慎考虑自己是否有错误的恋爱或择偶条件，这样会让你失去和对方进入深入关系的机会，当然也需要花时间来了解对方，以便最后决定是否确立恋爱关系。当然有特别信心等候神的具体旨意显明的情况除外。总之这一切都要清清洁洁的，本着圣经的原则，尤其是爱的原则进行，因为爱是最大的。而人一旦进入感情，最容易暴露的罪就是自私，自私，再自私，甚至在没有进入感情，在谈择偶或恋爱条件的时候就反映出来了。而我们最常见的罪就是不认罪。
 
　　如果有人上来对你说你是神为他预备的，你可以问他指的是什么意思，是神希望你和他进入婚姻/恋爱关系呢，还是神预定了你们的婚姻，你也可以让他分享他是怎么知道，看是否真的是神的作为。如果是后者，相当于神把未来一定要发生的事情告诉了他，相当于一个预言，也就是说你们早晚会进入婚姻的，谁也拦不住。神所命定发生的一定会发生。你只需要看结果就好了，如果最后你或他和别人结了婚，说明那个预言不是出于神，一定是假的。如果是前者，你需要详细了解对方寻求神的过程，然后凭你的信心确认是否出于神，然后再做决定。如同亚伯拉罕的仆人一路祷告寻求主赐给他以撒的妻子，当他把神带领他的整个过程，包括神对于他祷告的回应告诉利百加的家人的时候，得到了他们的确认，最后他们包括利百加都愿意顺服神，让利百加和老仆人回来做以撒的妻子。圣经中一个负面的例子是王上3章，一个老先知欺骗神人说天使奉耶和华的名让老先知带神人会他家吃饭，神人信以为真，就跟随老先知去了，最后违背了神的旨意，导致自己最后送了命。由此可见，在婚姻当中，神的旨意是最重要的，人的感情是相对次要的。在以撒和利百加成家之前，他们并没有时间谈恋爱，而是直接进入婚姻，然后才恋爱。这和一些人认为所谓的爱情是婚姻的基础形成鲜明对比。
 
　　其次，在你要和某人进入婚姻之前，你和对方都需要知道从圣经的角度婚姻到底是什么，意味着什么，有什么责任和角色，等等。这是负责任的态度，也是圣经的原则。因为你要在神面前为你的婚姻交账的。不要稀里糊涂地进入你一生中最重要的人际关系，然后后悔莫及。人生是不能走回头路的。婚姻是一个决定，是一生负责任的决定。当有一天你带着决心和你的爱人踏上红地毯的时候，你需要记住：婚姻是你人生新的一个阶段，和单身一样，有新的祝福，也有新的考验，你的安全感和依靠是在神那里，不是在人那里，当你不断学会在你的婚姻当中单单依靠神，单单相信神，单单服侍神，单单荣耀神的时候，你会不断学会享受你的婚姻，不断从婚姻中得自由，不断有超越婚姻的平安和喜乐，你会真心感受到：真正能给你带来祝福的，不是你当初选对了谁，而是你和那位爱你为你舍己的主的关系。他曾说“离了我，你们就不能做什么”（约15:5）。愿所有荣耀都归给他！
 
愿主祝福你们的恋爱和婚姻！
